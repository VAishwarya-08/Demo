# SauceDemo Appium Automation

Automated test suite for SauceDemo mobile web using Python, Appium, and Pytest.

##  Features
- Page Object Model (POM)
- Explicit waits
- Stable locators (ID/accessibility ID)
- HTML reporting with screenshots on failure

##Test Coverage
- Login (valid and invalid)
- Product catalog and details
- Add to cart and verify
-Sorting by price

## How to Run
1. Start Appium server
2. Connect Android device or emulator with Chrome installed
3. Run tests:
   ```bash
   pytest
